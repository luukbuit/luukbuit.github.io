# Hoe ziet een typische werkdag eruit?

- standup
- werken in sprints van 2 weken
- losse taken duren halve dag tot 5 dagen
- vaak samen met anderen
- 1 dag per week naar kantoor, rest thuis
- veel videobellen (2-4 uur per dag)
